# Deserts of Ajiibwan

A homebrew *D&D 5e* world, played as a campaign over ~5 years. That campaign has
ended; this folder supports **re-introducing the world to new players** — preserving
the accumulated lore and designing fresh adventures on top of it.

This is content for the [`dnd/`](../README.md) tooling; the scripts that operate on it
live in [`../tools/`](../tools/).

## What's here

| Folder | Contents |
|--------|----------|
| `assets/` | World lore converted from Google Docs/PDFs into committed markdown. `assets/INDEX.md` / `GLOSSARY.md` are generated by `tools/index.py`. `assets/_raw/` is git-ignored staging for the raw gdrive export. |
| `world/` | **The world compendium** — organized, cross-linked catalog of everything: [`gazetteer`](world/gazetteer.md) (places), [`adventures`](world/adventures.md), [`cast`](world/cast.md) (people), [`factions`](world/factions.md), [`artifacts`](world/artifacts.md), [`bestiary`](world/bestiary.md), [`lore-and-mysteries`](world/lore-and-mysteries.md), and the machine-readable [`entities.yaml`](world/entities.yaml) index. See [`world/README.md`](world/README.md). |
| `analysis/` | Human/AI reads of the corpus — `inventory.md` (what exists) and `story-structure.md` (the world + campaign spine). |
| `campaign/` | Live state of the new run: `state.yaml`, `threads.yaml`, `sessions/NN.md`, and `entities/` (npcs, locations, factions, monsters). |
| `scenarios/` | "What's next" adventure packets (`<name>.yaml`) rendered via `tools/render.py`. |
| `templates/` | Jinja2 templates for rendered GM packets and player handouts. |
| `build/` | Rendered PDFs and generated art (git-ignored). |

## Building the knowledge base

The source material lives in Google Drive (`gdrive-personal`), accessible only from the
**host OS**. See [`../CLAUDE.md`](../CLAUDE.md) for the host-side `rclone` export step and
the conversion/indexing workflow. In short:

```bash
# [HOST] export the campaign root into assets/_raw/
rclone copy "gdrive-personal:<CampaignRoot>" assets/_raw/ --drive-export-formats md

# [VM] from dnd/: convert and index
../../.venv/bin/python tools/convert.py ajiibwan
../../.venv/bin/python tools/index.py ajiibwan
```
