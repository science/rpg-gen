# Vendored fonts

These TrueType files are bundled so PDFs build offline. The **project code** is
Apache-2.0, but each font keeps its **own** license:

| Font | Role | License | Source |
|------|------|---------|--------|
| Limelight | Masthead display | SIL Open Font License 1.1 | [Google Fonts](https://fonts.google.com/specimen/Limelight) |
| Cinzel (Regular/SemiBold/Bold) | Section headers, labels | SIL Open Font License 1.1 | [Google Fonts](https://fonts.google.com/specimen/Cinzel) |
| EB Garamond (Regular/Italic) | Body text | SIL Open Font License 1.1 | [Google Fonts](https://fonts.google.com/specimen/EB+Garamond) |
| Special Elite | Typewriter fill-in fields | Apache License 2.0 | [Google Fonts](https://fonts.google.com/specimen/Special+Elite) |

The SIL OFL permits bundling and redistribution with this kind of work. Refresh
the files reproducibly with [`scripts/fetch-fonts.sh`](../../scripts/fetch-fonts.sh).
